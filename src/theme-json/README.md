# Theme.json
In this theme `theme.json` is compile from the `.mjs` files found in this directory. In order to affect changes in theme.json you'll need to edit the following files.
- `config.mjs`
- `settings.mjs`
- `styles.mjs`

See the Theme.json section of the [Gutenburg Handbook's Theme.json section under Best Practices](https://developer.bu.edu/gutenberg/best-practices/bu-child-starter/theme-json/) for how to work with the files and what setting are available for editing.
